
# WindowsでHHKB(US配列)を使ってMac風のキー操作を行うためのAutoHotkeyの設定

## 確認済み動作環境
- Windows 10

## 使い方
`as-mac-kana.ahk` をダブルクリック

## アンインストール
ファイルを削除

## できること
- ◇にCtrlを割り当てて
- 左右の◇空打ちでIMEのON/OFFを切り替え
- ◇+Tabでアプリケーションを切り替え
